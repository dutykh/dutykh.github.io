This time I would like to share another text that I started to write recently. In this document I decided to assemble various sources of information (books, papers) and also some scientific software libraries which turned out to be extremely useful in my scientific work. I can say it in other words: my vision of some fields of (Applied) Mathematics has been strongly influenced by these references. The PDF file can be found at this public Github repository:

* [https://github.com/dutykh/libs/](https://github.com/dutykh/libs/)

The reader will notice that most of my favourite references are rather old. It can be mainly explained by the personal taste of the author.

Please, note that it is a work in progress. So, it should be considered only as a beginning of a longer story. From time to time I shall add new elements upon the arrival.

I hope you will find some inspiration in the books which inspired me some time ago. Have a nice reading and, of course, any comments/remarks/propositions are welcome!
