Under the influence of some ideas about the [open science](http://www.openscience.org/blog/) and [reproducible research](http://www.siam.org/news/news.php?id=2064&goback=.gde_112393_member_232769759), I decided to start to release at [GitHub](https://github.com/) some code which could be useful for other researchers.

Here you can find two first releases containing the Matlab scripts that I previously used in my publications:

* [Okada solution](https://github.com/dutykh/Okada) implementation in Matlab to compute the static co-seismic displacements after an earthquake. This solution is widely used in *tsunami generation* problems
* [Babenko equation](https://github.com/dutykh/BabenkoSolitaryWave) solver in order to compute solitary wave solutions to the full Euler equations with the free surface

Some more codes will be released in the future. All my public repositories are regrouped under this URL address:  

* [https://github.com/dutykh](https://github.com/dutykh)
